# Udemy Clone - Media Queries Changes

## Objective
Improved the Udemy clone website by adding responsive design using CSS media queries.

## Features Added
- Responsive layout for mobile, tablet, and desktop
- Adjusted navbar for smaller screens
- Flexible course cards using Flexbox/Grid
- Images resized for different screen sizes
- Improved text alignment and spacing
- Better user experience on all devices

## Technologies Used
- HTML5
- CSS3
- Media Queries
- Flexbox

## Project Purpose
Practice responsive web design by making the Udemy clone adapt to different screen sizes.
